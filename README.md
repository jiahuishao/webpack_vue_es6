# es2015+vue组件化+webpack
> 生成package.json文件
```
npm init
```

 > 安装依赖
```
npm install --save-dev webpack

npm install --save-dev webpack-cli

npm install --save-dev babel-core 
    babel-loader@7 
    babel-preset-es2015 
    css-loader  
    style-loader 
    vue 
    vue-loader 
    vue-template-compiler
```

> 安装本地服务
```
npm install --save-dev webpack-dev-server
```
